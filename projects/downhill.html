Downhill

A 2D Casual game which player can play to pass some time can be a bit addictive if played for long.

Description

Downhill is my first official release through Google Play Store and its available for all to access now, the game is very simple player has to just tap on the enemies following while avoiding them hitting the bottom if they do it could cause player to loose a heart if they were to do the same for 3 times player would loose.

  • Genre: Casual
  • Platform: Android
  • Engine: Unity
  • Status: Released
