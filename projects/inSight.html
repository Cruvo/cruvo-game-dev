InSight

A 2D game where player has to shoot all the enmies in one bullet.

Description

InSight is a simple game that's made for Hack and Play game jam in 2 days. The game is simple, you just have to kill the enemies, but there is a catch you will only get one ammo and you have to kill the enemy in that one shot. Enjoy

  • Genre: Casual
  • Platform: Web (WebGL)
  • Engine: Unity
  • Status: Released
